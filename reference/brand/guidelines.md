# Brand guidelines — CTOC Command

CTOC is an internal-operations product brand, not a consumer/marketing one. The "brand" is the
operator aesthetic: calm, precise, dense, monochrome with disciplined severity color.

## Wordmark
- **CTOC · COMMAND**, set in the display face. In the topbar it appears as a square mark ("C")
  plus the wordmark; the mark alone is used on narrow screens.
- Clearspace: at least the mark's height on all sides. Don't recolor the mark with severity hues.

## Typography
- **Inter** (display), **DM Sans** (body), **JetBrains Mono** (data/labels). See
  [`../../design-system/foundations.md`](../../design-system/foundations.md) and
  [`typography/`](./typography/).

## Color
- Monochrome base (warm cream / warm charcoal); **severity is the only chromatic system**. The
  brand maps to product tokens in [`../../design-tokens.json`](../../design-tokens.json):
  ink/canvas for identity, `--sev-*` for state. No gradients-as-identity, no neon.

## Voice
- Operator-grade and terse. Full guidance:
  [`../../.claude/rules/content-voice.md`](../../.claude/rules/content-voice.md).

## Assets
- Logo files → [`logo/`](./logo/) · type specimens → [`typography/`](./typography/).
- Figma source of truth: file `i3YWhwkly7iJEmk0tK3YJ7`.

## Misuse
Don't: recolor the mark with status hues · stretch/skew the wordmark · place it on a busy image ·
introduce decorative gradients or saturated brand colors that compete with severity.
